# K3D Demo Windows Setup Notes

## Step P.1: GitBash

At Adligo we use GitBash to run just about everything, unless there is some reason not to use it.

- [Official GitBash Installation Instructions](https://git-scm.com/install/windows)

## Step P.2: GitBash

Note: On Windows I used the following command as administrator in a Windows Command Prompt to install curl & wget;

```
choco install wget
choco install curl
```

Then I could run wget or curl from GitBash.
